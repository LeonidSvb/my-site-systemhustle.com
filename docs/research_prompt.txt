### Critical sourcing notes:
- Job board postings (Indeed, LinkedIn, ZipRecruiter) often HIDE the facility/company name
  behind "confidential employer" — use them for signal type and specialty, not company name
- Careers pages can be stale (posted 6+ months ago) — always check publish date
- Trade press (Becker's, Modern Healthcare, Staffing Industry Analysts) = most reliable
- LinkedIn company pages: follower growth % is a real growth signal
- If you can't find 3 real verified examples — say so explicitly. Do NOT fabricate.

### How to use Exa results:
1. Read the full `text` field, not just the snippet
2. Look for: company name + signal type + recency (date published)
3. Cross-reference: if a company appears in 2+ separate searches = stronger signal
4. If facility name is hidden in a job posting — note "facility name not disclosed,
   signal verified via [source]" rather than guessing the name
